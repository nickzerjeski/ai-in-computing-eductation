# AI in Computing Education
The goal of this project is to create an educational video which respects the
Specifications and Assesment.

## Specifications
- 12-minute video seminar.
- Target: non-specialist audience.
- The content of the papers in `papers/` should be packaged in a meaningful,
clear and engaging way.
- The papers in `papers/` should only be the starting point which should be
complemented with other sources to
  - Clarify the context
  - Put the papers in perspective
  - Link it with other fields
- Style: modern and visually engaging animation

## Assesment
- **Clarity of communication**: The video message is clear and easy to
understand, using a style appropriate to the target audience.
- **Quality**: The video includes high-quality animation, audio, combination of
images and text, overall aesthetics.
- **Engagement**: The video keeps the audience interested, captures their
attention and maintains their interest troughout.
- **Congruence**: Connection between multiple pieces of information; Clear and
meaningful message based on a summary of multiple sources of information.
